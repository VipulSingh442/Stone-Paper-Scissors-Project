# 🎮 Stone Paper Scissors Game

A simple and interactive **Stone Paper Scissors** game built using **HTML, CSS, and JavaScript**. The user competes against the computer, and scores are tracked in real-time.

## 🚀 Features

* Interactive UI with images for Rock, Paper, and Scissors
* Random computer move generation
* Real-time score tracking
* Win/Lose/Draw result display
* Responsive and clean design
* Dynamic message colors based on game outcome

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

## 🎯 How to Play

1. Click on any of the three options:

   * 🪨 Rock
   * 📄 Paper
   * ✂️ Scissors

2. The computer will randomly choose its move.

3. Rules:

   * Rock beats Scissors
   * Scissors beats Paper
   * Paper beats Rock

4. The winner is displayed instantly and scores are updated.

## 🎮 Game Logic

1. User selects a move.
2. Computer randomly selects a move.
3. Both moves are compared.
4. Result is determined:

   * Win
   * Lose
   * Draw
5. Scores and messages are updated on the screen.
